City Updates

Project

September 2015 - December 2015

Description

City Updates is a web application created with the goal of making it easier for people to find out quick information on the current events of their desired city. This application was created for my final project for the course HCDE 310: Interactive Systems Design and Technology. Using various API's, this web application took the user's input and compiled various information onto one, easily accessible page.

Role: Developer

Tools: Python, HTML, CSS, Google App Engine, Wunderground's Weather API, Flickr API, New York Times' Article Search API
