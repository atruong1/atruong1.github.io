iLocalize

UX Designer

July 2016 - Current

Description

iLocalize is a web application game, designed to help users improve sound localization. With a team of three, we are working with the Swedish Medical Center, audiology department located in Seattle, WA, to get this app released to patients. I evaluated the user needs to determine key features necessary in the application and designed the UI and functionality of interactive objects on the application.

Team Members: Isaac Sek, Stephen Ramirez

Role: UX Designer

Tools: Github, Trello, HTML, CSS, and Adobe Photoshop
